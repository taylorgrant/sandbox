# MA.BABIP
Playing around with moving averages and player BABIP (batting average on balls in play) for the 2014 season. How do players fare over specific durations during the season and how do these periods compare to career means? Choose any player from 2014 for 
a given window. 

For instance, in 2014 Drew Stubbs had a BABIP of over .400. 
https://cloud.githubusercontent.com/assets/15227256/10899469/b8e633ae-8189-11e5-9c5c-4bfff3269581.png

Requires 2014 play-by-play data from retrosheet.org, included in the data folder.
